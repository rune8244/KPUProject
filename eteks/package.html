Instantiates the required classes to run Sweet Home 3D as a stand-alone application.

The main method of {@link com.eteks.homehiew3d.HomeView3D HomeView3D} class creates a {@linkplain com.eteks.homeview3d.model.HomeApplication home application} based on {@linkplain com.eteks.homeview3d.io.FileUserPreferences FileUserPreferences}, {@linkplain com.eteks.homeview3d.io.HomeFileRecorder HomeFileRecorder} and {@linkplain com.eteks.homeview3d.swing Swing views}.
